## 導入方法
#### 重要：データパックに対応するリソースパックを入れないと見た目が反映されません。また、データパックとリソースパックを入れる場所は同じではありません

###  ・データパック  
MInecraftにてデータパックを入れたいワールドを選択し、「編集」→「ワールドフォルダを開く」をクリックします。  
そこにある「datapacks」というフォルダに、こちらでダウンロードしたデータパックフォルダを入れます。  
(「〜_DP」という名前のフォルダが、データパックのフォルダです。)  

###  ・リソースパック  
Minecraftスタート画面の「設定...」→「リソースパック」→「パックフォルダーを開く」とクリックします。  
すると開くフォルダの中に、こちらでダウンロードしたリソースパックフォルダを入れます。  
(「〜_RP」という名前のフォルダが、データパックのフォルダです。)  
Minecraftに戻ると「利用可能」の場所にリソースパックが追加されているので、「選択中」に移動させましょう。  

これにて導入は完了です、お疲れ様でした。ぜひ楽しんで下さい。
